# BIA601 - Smart Algorithms Assignment (S25)

تطبيق ويب متكامل لتحسين توصيات متجر إلكتروني باستخدام **الخوارزمية الجينية (Genetic Algorithm)** اعتمادًا على ملف البيانات:
`HW__Data_S25` (الموجود هنا داخل مجلد `data_new`).

## تشغيل المشروع

```bash
npm install
npm start
```

ثم افتح:

`http://localhost:3000`

## المكونات

- `server.js`: خادم API وواجهة الموقع.
- `src/dataLoader.js`: تحميل وتهيئة بيانات Excel.
- `src/recommender.js`: منطق التوصية + التحسين الجيني.
- `public/`: واجهة ويب عربية تعرض النتائج.
- `REPORT.md`: التقرير الأكاديمي المطلوب.

## نقاط مهمة للوظيفة

- التطبيق يجيب عمليًا على:
  - ما معنى تحسين التوصيات؟
  - كيف تحسن الخوارزمية الجينية جودة التوصيات؟
- تم استخدام مقال علمي حديث ضمن المدة المطلوبة 2024-2026.
- كل الملفات جاهزة للرفع على Git.
